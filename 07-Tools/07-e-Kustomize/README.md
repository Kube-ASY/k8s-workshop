# Kustomize / Kustomization

## Object Generator
* Ordner `app`
  * deployment.yaml
  * service.yaml
  * vhost.conf
  
* In der kustomization.yaml wird aus der vhost.conf eine ConfigMap generiert:

```yaml
configMapGenerator:
- name: nginx-config
  files:
  - vhost.conf
resources:
- nginx-deployment.yaml
- nginx-service.yaml
```

* Testing

  ```bash
  kubectl kustomize app
  ```
