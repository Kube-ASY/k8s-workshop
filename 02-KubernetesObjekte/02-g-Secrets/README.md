## Usage as Volume Mount

```yaml
spec:
  containers:
  - image: nginx:1.19
    name: nginx
    volumeMounts: 
      - name: auth-volume
        mountPath: /mnt/nginx-auth
  volumes:
    - name: auth-volume
      secret:
        secretName: nginx-auth
```
